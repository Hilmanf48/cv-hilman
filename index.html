Tentang saya

Saya adalah seorang mahasiswa aktif di Universitas Islam Indonesia, Yogyakarta, jurusan Teknik Informatika. Saya memiliki ketertarikan yang besar dalam bidang data science dan machine learning. Saya juga memiliki pengalaman dalam mengembangkan aplikasi berbasis web dan mobile.

Saya memiliki kemampuan dalam bahasa pemrograman Python, R, dan SQL. Saya juga terbiasa menggunakan berbagai library dan framework seperti Pandas, NumPy, Scikit-learn, TensorFlow, dan Keras. Selain itu, saya juga memiliki pengalaman dalam menggunakan alat-alat seperti Git, Docker, dan Jupyter Notebook.

Saya memiliki kemampuan analisis yang baik dan mampu bekerja dalam tim maupun secara mandiri. Saya juga memiliki kemampuan komunikasi yang baik dan mampu menyampaikan ide-ide saya dengan jelas.

Saya sangat antusias untuk belajar dan mengembangkan diri dalam bidang data science. Saya percaya bahwa dengan kerja keras dan dedikasi, saya dapat mencapai tujuan saya dan memberikan kontribusi yang berarti bagi perusahaan.
